# Dockerized Java Web App with Nginx, Tomcat, RabbitMQ, Memcached, Elasticsearch, and MySQL

# Project Overview

This project demonstrates the containerization of a Java web application using Docker. It integrates key production-grade services including:

- **Nginx**: Reverse proxy and load balancing
- **Tomcat**: Java web server to deploy `.war` file
- **MySQL**: Database
- **RabbitMQ**: Messaging queue for async processing
- **Memcached**: Caching layer
- **Elasticsearch**: Search engine and log indexing
- **Docker**: Containerization and environment consistency

All services are containerized using Docker, making the system portable, scalable, and easy to deploy.
